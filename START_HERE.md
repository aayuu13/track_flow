## 🚀 Next Steps (In Order)

### Step 1: Firebase Setup (10 min)
1. Go to [Firebase Console](https://console.firebase.google.com)
2. Create new project
3. Enable Firestore Database
4. Enable Email/Password Authentication
5. Copy your Firebase config
6. ⏱️ **Time: ~10 minutes**

### Step 2: Add Credentials (2 min)
1. Open `.env.example` file
2. Copy it to `.env.local`
3. Paste your Firebase keys
4. Save the file
5. ⏱️ **Time: ~2 minutes**

### Step 3: Start the App (1 min)
```bash
npm run dev
```
App opens at `http://localhost:3000` automatically!
⏱️ **Time: ~1 minute**

### Step 4: Test It! (5 min)
1. Click "Sign Up"
2. Create test account
3. Add a product
4. Create a transaction
5. Check reports
⏱️ **Time: ~5 minutes**
